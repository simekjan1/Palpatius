První vydání: Backup Protocol Tools (offline)

- Kompletní offline aplikace pro sledování záložních zařízení a stavů zálohování
- Data uložená lokálně pomocí IndexedDB
- Export do Markdownu pro lidsky čitelné záznamy
- Plný export/import JSON pro zálohování a obnovu
- Žádné externí knihovny, fonty ani síťové závislosti

Projekt je záměrně dokončen.
Pokud chcete změny nebo vylepšení, můžete jej přizpůsobit svým vlastním potřebám.
