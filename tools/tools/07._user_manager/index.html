Palpatius User Manager

Tento nástroj slouží k vytvoření jednoduchého identifikačního souboru, kterým si můžeš označit tuto lokální verzi systému Palpatius jako svou.

Vyplň jen to, co chceš. Žádné pole není povinné.

Údaje o uživateli

Základní údaje
Firma / masérna (nepovinné)
Poznámka

Připraveno.

Jak to funguje

  • Vyplníš libovolné údaje (klidně jen jedno pole).
  • Klikneš na Vytvořit soubor.
  • Stáhne se soubor palpatius-user.md.
  • Tento soubor si uložíš do hlavní složky Palpatius.

Nástroj nic neukládá, nic neodesílá a do systému Palpatius nijak nezasahuje.
